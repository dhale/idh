Mines Java Toolkit (JTK)

Open-source Java packages for science and engineering.

#content()

The Mines Java Toolkit (JTK) is a set of Java packages and native (non-Java) code libraries for science and engineering. Applications currently include digital signal processing and 2-D and 3-D graphics.

The Mines JTK is open-source software made available under the terms of the Common Public License - v 1.0.

You may browse documentation online for all Java packages, classes, and methods in the Mines JTK.

Source code is our most up-to-date and least ambiguous documentation, and you can browse our source code online. There you will also find information on building and using the Mines JTK.

#sidebar()

Standard Tools

To build and use the Mines JTK, you need only the

  • Java Development Kit (JDK), which includes the Java compiler and runtime environment, and
  • Gradle, which is a portable system for building software.

These tools are widely used among Java software developers. Complete instructions are here.
